ChallengeDTO Up

Represents a challenge with its details.
name
String Name of the challenge.
example: Marathon Training
startDate
date Start date of the challenge. Must be in the future or present. format: date
example: Thu Jan 01 00:00:00 GMT 2026
endDate
date End date of the challenge. Must be after or equal to the start date. format: date
example: Sat Jan 31 00:00:00 GMT 2026
objectiveValue
Double Objective value for the challenge (e.g., distance in km). format: double
example: 42.2
objectiveType
String Type of the objective for the challenge.
Enum:
TIEMPO
DISTANCIA
example: DISTANCIA
sport
String Sport type of the challenge.
Enum:
RUNNING
CICLISMO
example: RUNNING

FilterDTO Up

Filter criteria for searching challenges or training sessions.
startDate (optional)
date Start date for the filter. Challenges starting on or after this date will be included. format: date
example: Mon Jan 01 00:00:00 GMT 2024
endDate (optional)
date End date for the filter. Challenges ending on or before this date will be included. format: date
example: Tue Dec 31 00:00:00 GMT 2024
sport (optional)
String Type of sport for the filter. Only challenges of this sport type will be included.
Enum:
RUNNING
CICLISMO
example: CICLISMO
limit (optional)
Integer Maximum number of challenges to retrieve. format: int32
example: 10
